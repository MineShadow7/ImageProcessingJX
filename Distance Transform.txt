1. Подготовка изображения: Сначала необходимо подготовить изображение,
на котором вы хотите выполнить сегментацию. Обычно это черно-белое изображение,
где объекты представлены белыми пикселями, а фон - черными.
Если у вас есть цветное изображение, вы можете сначала преобразовать его в черно-белое изображение.

2. Бинаризация изображения: Затем следует применить бинаризацию для преобразования изображения
 в бинарное, где пиксели объектов имеют значение 1, а фон - значение 0.
 Это важно для дальнейшей обработки.

3. Выполнение Distance Transform: Алгоритм Distance Transform вычисляет расстояние
от каждого пикселя до ближайшей граничной области.
Для этого применяется итеративный процесс,
в котором каждый пиксель обрабатывается отдельно.
На каждой итерации алгоритм вычисляет расстояние текущего пикселя от всех соседних пикселей
и выбирает минимальное значение.
Затем полученное расстояние присваивается пикселю.
Этот процесс продолжается до тех пор, пока все пиксели не будут обработаны.

4. Бинаризация Distance Transform: После выполнения Distance Transform полученное изображение
может иметь разные значения для расстояний. Часто требуется преобразовать его обратно в
бинарное изображение.
Для этого можно выбрать пороговое значение и применить бинаризацию,
где пиксели с расстоянием, превышающим порог, считаются границей объектов,
а остальные пиксели - фоном.
